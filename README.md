# Packet Drop Simulator

A simple simulation tool for modeling packet loss in a network environment. This project helps visualize how unreliable connections affect data transmission.

## 🚀 Features

* Simulates packet transmission between sender and receiver
* Configurable packet drop rate
* Logs successful and dropped packets
* Useful for learning networking concepts and testing reliability strategies
